Globe Forecast is a weather application that allows users to search any city around the world to view real-time weather conditions, including temperature, humidity, and wind speed.
The app also provides a 7-day forecast with the maximum and minimum temperatures for each day. Featuring animated weather icons for a dynamic user experience, the app is powered by a weather API to ensure up-to-date and accurate information for global locations.

WORLD CLOCK APP

Time-Loop is an interactive world clock that allows users to select multiple countries, toggle between 12-hour and 24-hour formats, and view real-time hours, minutes, and seconds.
The clock includes a feature to display the current local time of the user's selected location, with some countries fixed for quick access.
It offers a responsive, user-friendly interface that adapts to different time zone needs, making it an ideal tool for global time tracking.
